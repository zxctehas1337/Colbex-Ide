# Colbex - Product Overview

Colbex is a cross-platform code editor/IDE built with Tauri 2.0. It provides a VS Code-like experience with integrated AI assistance, Git support, and a modern React-based UI.

## Core Features

- **Code Editor**: Monaco-based editor with syntax highlighting, multi-tab support, and breadcrumb navigation
- **AI Assistant**: Multi-provider AI chat (OpenAI, Anthropic, Google, xAI, Ollama) with agent and responder modes
- **File Explorer**: Tree-based file browser with file watching, search, and workspace management
- **Git Integration**: Built-in Git support for staging, commits, branches, and diffs
- **Terminal**: Integrated terminal panel with port monitoring
- **NPM Scripts**: NPM script runner with process management
- **Timeline**: File history and snapshot management
- **Theming**: Multiple themes (Dark Modern, Dracula, GitHub Dark, Nord, One Dark, etc.)

## Target Platforms

- Windows (MSI installer)
- Linux (DEB, AppImage)
- macOS (planned)
